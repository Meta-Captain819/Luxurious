## ğŸ“¦ Getting Started

To run the project locally:

1. **Clone the repository:**

   ```bash
   git clone https://github.com/Meta-Captain819/Luxurious.git
   cd Luxurious
   ```

2. **Install dependencies:**

   ```bash
   npm install
   # or
   yarn install
   ```

3. **Start the development server:**

   ```bash
   npm run dev
   # or
   yarn dev
   ```

4. **Open your browser and navigate to:**

   ```
   http://localhost:3000
   ```
